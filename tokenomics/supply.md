# Supply

### What is FRAC?

FRAC is the native token of the Fractal network. It is used to incentivise network participants to act honestly and fulfil network responsibilities.

### FRAC properties

| Property       | Value              |
| -------------- | ------------------ |
| Token          | FRAC               |
| Initial Supply | 1,000,000,000 FRAC |
| Decimals       | 10                 |
